Plugin for ad supported syndication of articles

== Description ==

The Embed Article wordpress plugin allows blog owners to syndicate their articles that contain traveling ad tags on 3rd party sites.  For more information visit: http://embedarticle.com/how-it-works

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `embedarticle.zip` to the `/wp-content/plugins/` directory
2. Unzip `embedarticle.zip`
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Visit Settings -> Embed Article Options to Setup

== Frequently Asked Questions ==

= Do I need an account on embedarticle.com to use this plugin? =

Yes. In order to complete the setup of the plugin you need a Publisher ID from embedarticle.com.  Please use this link to sign up:

http://embedarticle.com/register

It is fast and free.  Once you are registered, visit the Wordpress page under the Integrate section of the site.
